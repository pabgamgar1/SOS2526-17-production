# SOS2526-17

Minimal SvelteKit + Express app with three public APIs and a matching frontend.

## Run

```bash
npm install
npm start
```

## Routes

- `/`
- `/api/v1/water-productivities`
- `/api/v1/agriculture-land`
- `/api/v1/renewable-energy-consumptions`
